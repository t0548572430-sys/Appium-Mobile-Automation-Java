# Appium Mobile Automation Project

This project contains automated tests for Android system applications using **Appium**, **Java**, and **JUnit 5**.

## Tasks Included:
1. **Clock App Test**: Sets a 30-second timer, waits for it to finish, and verifies the reset functionality.
2. **Settings App Test**: Navigates through System settings, scrolls to find "About device" using a Scrollable UISelector, and verifies the page content.

## Technologies Used:
- Java
- Appium (UiAutomator2)
- Selenium WebDriver
- JUnit 5
- Maven
